# HYTRIX — Premium Digital Agency Website

A fully responsive, single-page agency website built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just drop and deploy.

---

## 🚀 Deploy to Vercel (Recommended)

### Option 1 — One-click via Vercel Dashboard

1. Push this repo to GitHub (see below)
2. Go to [vercel.com](https://vercel.com) and click **"Add New Project"**
3. Import your GitHub repo
4. Leave all settings as default — Vercel auto-detects static sites
5. Click **Deploy** ✅

Your site will be live at `https://your-project.vercel.app`

### Option 2 — Vercel CLI

```bash
npm i -g vercel
vercel
```

Follow the prompts. Done.

## 🌐 Custom Domain on Vercel

1. Go to your project on vercel.com → **Settings → Domains**
2. Add your domain (e.g. `hytrix.in`)
3. Update your domain's DNS with the records Vercel provides
4. Done — SSL is automatic ✅
